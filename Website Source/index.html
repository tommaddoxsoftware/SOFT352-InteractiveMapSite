Christmas Tree Locator

What is it?

This piece of software allows employees to quickly locate trees that have been chosen by customers to be cut down for them this Christmas.

The software allows employees to view details such as the height of the tree, the colour of the label, and the date of which it should be cut on.

How do I use It?

To access the map, Scroll down to the Map. This will have loaded all customer trees and placed them on the map with a map marker.

To view details for each tree, click on it's marker. Here you will be shown relevant information for that tree. To close the information panel, click on the "x" icon in the top right corner.

If you are logged in, you will be able to use the edit functions of the software to make changes to tree information. You'll need your randomly generated 4 digit login pin to do so.
